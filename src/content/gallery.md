# Gallery

Explore interactive examples showcasing Hex1b's capabilities. Each demo runs live in your browser, connected to a real .NET backend via WebSocket.

<TerminalGallery />

::: tip Running Locally
To run the gallery locally, start the Hex1b.Website project:
```bash
dotnet run --project src/Hex1b.Website
```
Then open the VitePress dev server with `npm run dev`.
:::
