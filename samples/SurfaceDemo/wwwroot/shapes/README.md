# Smart Matter Shapes

Place monochrome silhouette images (PNG, JPG, BMP) in this directory to use them in the Smart Matter demo.

## Image Requirements

- **Format**: PNG, JPG, JPEG, or BMP
- **Colors**: Dark pixels (grayscale < 128) become target positions for particles
- **Size**: Any size - images are automatically scaled to fit the terminal's braille resolution
- **Style**: Simple silhouettes work best (solid black shapes on transparent/white backgrounds)

## Recommended Sources

For royalty-free silhouette images, consider:

- [OpenClipart](https://openclipart.org/) - Public domain vector clipart
- [Pixabay](https://pixabay.com/) - Free images (check license)
- [Unsplash](https://unsplash.com/) - Free photos (check license)
- [SVG Silhouettes](https://freesvg.org/) - Free SVG files (convert to PNG)

## Notes

If no images are found in this directory, the demo uses built-in procedural shapes:
- Heart
- Star
- Circle
- Tree
- Rocket
